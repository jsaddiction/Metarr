# Core UI Component Examples

These examples demonstrate how to use the new core UI components created as part of the styling architecture refactor.

## PageContainer

Wraps page content with consistent title, optional subtitle, and proper spacing.

```tsx
import { PageContainer } from '@/components/ui/PageContainer';

function MyPage() {
  return (
    <PageContainer 
      title="Page Title" 
      subtitle="Optional descriptive subtitle"
    >
      {/* Your page content */}
    </PageContainer>
  );
}
```

## SettingCard

Creates a raised card with title, optional description and icon, for grouping related settings.

```tsx
import { SettingCard } from '@/components/ui/SettingCard';
import { Settings } from 'lucide-react';

function MySettings() {
  return (
    <SettingCard 
      title="Section Title"
      description="Optional description of this section"
      icon={<Settings className="w-5 h-5" />}
      variant="default" // or "subtle" for semi-transparent background
    >
      {/* Setting rows go here */}
    </SettingCard>
  );
}
```

## SettingRow

Creates a row with label, optional description, and a control element.

```tsx
import { SettingRow } from '@/components/ui/SettingRow';
import { Switch } from '@/components/ui/switch';

function MySetting() {
  return (
    <SettingRow 
      label="Enable Feature"
      description="Optional description of what this setting does"
    >
      <Switch checked={enabled} onCheckedChange={setEnabled} />
    </SettingRow>
  );
}
```

## SectionStack

Provides consistent vertical spacing between sections.

```tsx
import { SectionStack } from '@/components/ui/SectionStack';

function MyPage() {
  return (
    <SectionStack spacing="default"> {/* or "compact" for tighter spacing */}
      <SettingCard title="First Section">...</SettingCard>
      <SettingCard title="Second Section">...</SettingCard>
      <SettingCard title="Third Section">...</SettingCard>
    </SectionStack>
  );
}
```

## Complete Example

Here's how these components work together to replace inline Tailwind:

```tsx
import { PageContainer } from '@/components/ui/PageContainer';
import { SectionStack } from '@/components/ui/SectionStack';
import { SettingCard } from '@/components/ui/SettingCard';
import { SettingRow } from '@/components/ui/SettingRow';
import { Switch } from '@/components/ui/switch';
import { Settings, Zap } from 'lucide-react';

function SettingsPage() {
  const [enabled, setEnabled] = React.useState(false);
  const [advanced, setAdvanced] = React.useState(false);

  return (
    <PageContainer 
      title="Settings" 
      subtitle="Configure your application preferences"
    >
      <SectionStack>
        <SettingCard 
          title="General Settings" 
          icon={<Settings className="w-5 h-5" />}
        >
          <SettingRow 
            label="Enable Feature"
            description="Turn this feature on or off"
          >
            <Switch checked={enabled} onCheckedChange={setEnabled} />
          </SettingRow>
        </SettingCard>

        <SettingCard 
          title="Advanced Settings" 
          icon={<Zap className="w-5 h-5" />}
          variant="subtle"
        >
          <SettingRow 
            label="Advanced Mode"
            description="Enable advanced features"
          >
            <Switch checked={advanced} onCheckedChange={setAdvanced} />
          </SettingRow>
        </SettingCard>
      </SectionStack>
    </PageContainer>
  );
}
```

## CSS Utility Classes

These utility classes are now available globally in `globals.css`:

### Card Classes
- `.card-raised` - Standard raised card (bg-neutral-800)
- `.card-raised-subtle` - Semi-transparent raised card (bg-neutral-800/50)

### Layout Classes
- `.page-container` - Page wrapper with content spacing
- `.page-header` - Page header section
- `.page-title` - Page title text (2xl, semibold, white)
- `.page-subtitle` - Page subtitle text (sm, neutral-400)

### Form Classes
- `.setting-row` - Flex row for settings (space-between)
- `.input-standard` - Standard height input (h-10)
- `.input-compact` - Compact height input (h-8)

### Spacing Classes
- `.section-stack` - Default vertical spacing (space-y-6)
- `.section-stack-compact` - Compact vertical spacing (space-y-3)

## Design Tokens

New CSS variables added to `@theme`:

```css
/* Surface Colors */
--color-surface-app: #171717;
--color-surface-raised: #262626;
--color-surface-overlay: #404040;

/* Borders */
--color-border-default: #404040;
--color-border-subtle: #2626267f;

/* Spacing */
--spacing-card: 1.5rem;
--spacing-compact: 0.75rem;
--spacing-section: 1.5rem;

/* Sizing */
--input-height-standard: 2.5rem;
--input-height-compact: 2rem;

/* Radius */
--radius-card: 0.75rem;
```

These tokens can be used directly in component styles or referenced in utility classes.
